# myHome

![startpage](public/images/demo.jpg)

This is a platform for home ads and home search. It's an unfinished project I made to test a few things out, yet most of what's necessary for such a platform is already implemented, including sophisticated features such as search and filtering, user authentication, and a few other things.

## Tech stack

- [Vite](https://vitejs.dev/)
- [Vue 3](https://v3.vuejs.org/)
- [Vue Router](https://next.router.vuejs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
